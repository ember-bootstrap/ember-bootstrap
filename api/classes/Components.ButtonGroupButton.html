onClick

public

Inherited from Components.Button: addon/components/bs-button.js:390

When clicking the button this action is called with the value of the button (that is the value of the "value" property). Return a promise object, and the buttons state will automatically set to "pending", "resolved" and/or "rejected". This could be used to automatically set the button's text depending on promise state (defaultText, pendingText, fulfilledText, rejectedText) and for futher customization using the yielded isPending, isFulfilled, isRejected properties. The click event will not bubble up, unless you set bubble to true.
