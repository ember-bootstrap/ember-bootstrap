Carousel

A slideshow component for cycling through elements.

This page shows some example usage of the <BsCarousel> component. For the full documentation please refer to the API Docs. You might also want to consult the official Bootstrap Docs for general information.
Previous Next 
<BsCarousel
  @autoPlay={{true}}
  @wrap={{true}}
  @index={{0}}
  @interval={{5000}}
  @keyboard={{true}}
  @ltr={{true}}
  @pauseOnMouseEnter={{true}}
  @showControls={{true}}
  @showIndicators={{true}}
  as |car|>
  {{#each slides as |slide|}}
    <car.slide>
      <img alt={{slide.alt}} src={{slide.src}}>
    </car.slide>
  {{/each}}
</BsCarousel>
